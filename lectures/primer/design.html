Vague but Exciting…

HTML — the design of the web was invented by, one person, Tim Berners-Lee circa 1989. The idea of using tags (words between angle brackets) already existed in SGML (Standard Generalized Markup Language). Rather than inventing an entirely new standard, Tim Berners-Less built upon existing markup to create HTML, similar to today’s development of HTML5.
